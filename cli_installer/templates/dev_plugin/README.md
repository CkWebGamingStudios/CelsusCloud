# 🧩 Celsus Plugin Template

This is the official development template for building Celsus plugins.

## 🚀 Commands
- `celsus install dev <path>` → sets up this structure at your location.

## 📂 Structure
- `src/` → main logic entry (`index.js`)
- `public/` → frontend assets
- `command.js` → plugin command definitions
- `manifest.json` → plugin metadata

You can zip this folder and publish it using `celsus-publish.mjs`.
